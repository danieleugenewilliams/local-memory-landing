## Core Capabilities

**Memory Management (4 MCP Tools)**
- store_memory: Create memories with content, tags, importance, domain
- update_memory: Modify existing memory content and metadata
- delete_memory: Remove memories with relationship cleanup
- get_memory_by_id: Retrieve specific memory details

**Search & Analysis (2 MCP Tools)**
- search: Multi-mode search (semantic, tags, date_range, hybrid) with 10-57ms performance
- analysis: AI-powered analysis (question, summarize, analyze, temporal_patterns)

**Organization (3 MCP Tools)**
- relationships: Memory connections and graph mapping
- categories: Hierarchical memory organization
- domains: Knowledge domain management

**Administration (2 MCP Tools)**
- stats: Comprehensive statistics and metrics
- sessions: Session management and tracking

## REST API Highlights

**Base URL**: http://localhost:3002/api/v1

**Memory Operations (9 endpoints)**
- POST /memories: Create new memory
- GET /memories: List with pagination
- POST /memories/search: Enhanced search with optimization
- GET /memories/:id: Retrieve specific memory
- PUT /memories/:id: Update existing memory
- DELETE /memories/:id: Remove memory

**AI Analysis (3 endpoints)**
- POST /analyze: Pattern analysis and insights
- POST /ask: Question answering with context
- POST /summarize: Memory summarization

**Advanced Features**
- Temporal analysis for learning progression
- Relationship discovery and mapping
- AI-powered categorization
- Response format optimization (detailed/concise/ids_only/summary)
